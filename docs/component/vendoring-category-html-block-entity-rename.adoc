= Vendoring CategoryHtmlBlock Entity Rename

This patch removes the generic `App\Vendoring\Entity\CategoryHtmlBlock` runtime class from `Vendoring`.

== Refactor

----
src/Entity/CategoryHtmlBlock.php
----

is renamed to:

----
src/Entity/VendorCatalogCategoryHtmlBlock.php
----

and the class is renamed from `CategoryHtmlBlock` to `VendorCatalogCategoryHtmlBlock`.

== Reason

The combined App Kernel cannot load multiple components that declare the same FQCN:

----
App\Vendoring\Entity\CategoryHtmlBlock
----

Both Cataloging and Vendoring now use component-scoped class names.
